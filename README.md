# NEWLIFE CHRISTIAN ACADEMY LIBRARY MANAGEMENT SYSTEM

A web-based library management system designed for **Newlife Christian Academy**.  
This project helps manage books, track borrowing and returning, and streamline library operations for students and staff.

## 🚀 Features
- Book catalog with search functionality
- Borrow and return tracking
- Admin panel for managing books and users
- User-friendly interface with responsive design
- Secure and organized data management
